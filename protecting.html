Protection - World Guard

Name Command Description
Selecting the area Select the area using world edit. Use this to define the region. Do this first.
Defining the region /region define [areaName] Use this to create an area with default settings
Adding an owner /region addowner [areaName] [ownerName] Use this to add owners to the area
Stopping people entering the area /region flag entry [areaName] [allow|deny] Stops/Allows non-owners entering the area (Default - Allow)
Stopping mob spawning in the area /region flag mob-spawning [areaName] [allow|deny] Stop/Allow mobs to spawn in the area (Default - Allow)
Allowing PvP /region flag pvp [areaName] [allow|deny] Allows/Denys PvP in an area (Default - Deny)
